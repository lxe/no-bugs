A JavaScript test framework for perfect code.

 - Lightning fast
 - Seamless reporting
 - Always covers 100% of your code

### Installation

Install it via `npm`:

```
npm install --save-dev no-bugs
```

Adjust "test" script in your `package.json`:

```
...
"scripts": {
  "test": "no-bugs"
}
...
```
